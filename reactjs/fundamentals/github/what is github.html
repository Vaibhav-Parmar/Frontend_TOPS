What is github ?

Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently. Git is used to tracking changes in the source code, enabling multiple developers to work together on non-linear development.7

How to upload Data on github server ?

  1. create an account on github.com
  2. download git software to upload data on github
    https://git-scm.com/downloads/win
  3. create a repository on github
  4. git init
  5. git add .
  6. git commit -m "16-12-2024 all data uploaded"
  7. git remote add origin https://github.com/Brijesh1990/nodejs-12-mwf.git
  8. git push -u origin main
